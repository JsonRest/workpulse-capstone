# Data — Raw

Raw CSV files are generated by the notebooks (self-contained).

To use real Kaggle datasets instead, download from:
- https://www.kaggle.com/datasets/pavansubhasht/ibm-hr-analytics-attrition-dataset
- https://www.kaggle.com/datasets/blurredmachine/are-your-employees-burning-out
- https://www.kaggle.com/datasets/waqi786/workplace-stress-and-mental-health-dataset

Place the CSVs in this directory and uncomment the `pd.read_csv()` lines in the notebooks.
