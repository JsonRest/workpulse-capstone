# Models

Saved model artefacts are generated by Step 4 notebook:
- `best_model.pkl` — XGBoost (Tuned), the selected production model
- `*_model.pkl` — All 13 trained models (joblib)
- `mlp_model.keras` — MLP neural network (TensorFlow)
- `scaler.pkl` — StandardScaler fitted on training data
- `feature_columns.pkl` — List of 13 selected feature names

To regenerate: run `notebooks/04_model_implementation.ipynb` or:
```bash
python -m src.train --model xgboost --tune --output models/
```
